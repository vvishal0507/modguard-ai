# ModGuard AI

**AI-Powered Moderation Assistant for Safer Online Communities**

## About

ModGuard AI is a simple AI-based moderation assistant designed to help online community moderators review posts and comments faster. It can identify possible spam, toxic language, rule-breaking content, and posts that may need manual review.

This project is built for a hackathon idea to support safer and healthier online communities.

## Features

- Detects possible spam content
- Flags toxic or abusive comments
- Shows content status such as Safe, Spam, Toxic, or Needs Review
- Gives simple moderation suggestions
- Clean and beginner-friendly dashboard UI

## How It Works

1. User enters a post or comment.
2. The system analyzes the text.
3. It displays a moderation result.
4. Moderator can review the suggestion and take action.

## Project Goal

The goal of ModGuard AI is not to replace human moderators, but to support them by making moderation faster, clearer, and safer.
